\# LO-3DGS Viewer



Interactive Gaussian Splat viewer built on LightOrigin.



\## Goals



\- simple embedding

\- clean API

\- reusable viewer

\- configurable through settings.json

\- no framework dependency

\- lightweight



\## Main technologies



\- JavaScript

\- LightOrigin Viewer

\- Three.js

\- GSAP

\- Vite
